Sending request to telegram. <br/> Only telegram method can be invoked.

Parameters

  • Telegram client.
🔗

  • Raw class from telegram method.
🔗

  • Max retries for invoking. default is same with ClientInterface.maxRetries or 5.
🔗

  • How long to wait for the function to finish. default is 15s.
